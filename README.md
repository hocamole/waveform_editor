# Waveform Grid Editor

디지털 회로 **파형(waveform)** 을 격자 위에서 쉽게 그리고, 그 결과를 **이미지로 캡처**하거나 **텍스트(WaveDrom / 자체 JSON)** 로 변환해 주는 단일 HTML 페이지입니다.

> Visio·엑셀로 파형을 그리는 것보다 빠르게 그리고, AI 학습·RAG용으로 쓰기 좋은 텍스트 표현까지 한 번에 얻는 것이 목표입니다.
 
- **의존성 없음** — `index.html` 파일 하나. 인터넷/설치 없이 더블클릭으로 실행됩니다.
- 웹 접속이 더 편하다면 https://hocamole.github.io/waveform_editor/index.html 에서 사용할 수도 있습니다.
- **두 가지 용도**
  1. 파형을 빠르게 그리는 **에디터** (+ 깔끔한 캡처 모드)
  2. 파형을 **텍스트로 변환** (WaveDrom 문법 + 무손실 자체 JSON)

---

## 시작하기

1. `index.html` 을 브라우저로 엽니다(더블클릭).
2. 위쪽 **툴바**에서 그릴 값/도구를 고르고, 가운데 **격자**에서 파형을 그립니다.
3. 아래쪽 패널에서 **WaveDrom 텍스트**(자동 생성)와 **자체 JSON**(저장/불러오기)을 확인합니다.

> 권장 브라우저: **Chrome / Edge** (그 외 브라우저에서는 「JSON 저장」이 탐색기 창 대신 일반 다운로드로 동작합니다.)

## 화면 구성

격자의 **행 = 신호(signal)**, **열 = 시간 스텝(time step)** 입니다.

### 툴바 (3줄)

| 줄 | 그룹 | 내용 |
|----|------|------|
| 1 | **값** | `1/0` · `D` · `z` · `x` · `지움` (셀 값 펜) · `전체 지움` |
| 1 | 동작 | `캡처 모드` · `캡처`(JPG 이미지 저장) |
| 1 | 변환 | `JSON 생성` · `JSON 저장` · `JSON 열기` |
| 2 | **데이터** | `배경`색 · `글자`색 · `적용` |
| 2 | 크기 | 셀 `W`/`H` · `신호 H`(파형 진폭) |
| 2 | 그리드 | `행`·`열` 개수 · `선 굵기` |
| 3 | **주석** | `화살표` · `양방향 화살표` · `점선` · `사각형` · `원/타원` · `T`(텍스트) · `이동` · `삭제` |
| 3 | 주석 색 | `선`색 · `글자`색 · `적용` |

## 파형 그리기

값 펜을 고른 뒤 격자 셀을 클릭/드래그합니다.

| 펜 | 의미 | 사용법 |
|----|------|--------|
| **1/0** | 1비트 신호 (검정 실선) | 셀 **위쪽 클릭 = High**, **아래쪽 = Low**. 드래그로 연속 그리기. |
| **D** | 데이터 버스 | **드래그하면 미리보기**, 놓는 순간 확정. **더블클릭으로 값 입력**(빈칸 기본, 줄바꿈 X). |
| **z** | High-Z | 가운데 **빨간 실선**. |
| **x** | Unknown | **데이터 버스 모양** + 내부 **검정 사선 해치**. |
| **shot** | 임펄스 | **0 레벨 가로선** + **왼쪽 경계의 굵은 세로선**(선 굵기 +1px). WaveDrom에는 `1`로 표기. |
| **지움** | 셀 비우기 | 클릭/드래그한 셀을 빈칸으로. |

- **데이터 버스 겹침**: 새 버스를 기존 버스 위에 그리면, 새 버스가 그 영역을 차지하고 기존 버스는 그만큼 줄어듭니다(가운데를 침범하면 둘로 나뉨).
- **데이터 색 지정**: `배경`/`글자` 색을 고른 뒤 **`적용`** 을 눌러 버스를 클릭하면 그 버스에만 색이 칠해집니다. 색을 바꿔도 **기존 버스는 그대로**이고, 새로 그리는 버스에만 현재 색이 반영됩니다.

### 신호(행) 다루기

- **이름 편집**: 왼쪽 신호 이름 칸을 클릭해 입력. (이름 칸은 가장 긴 이름에 맞춰 자동으로 넓어집니다.)
- **순서 변경**: 이름 칸의 손잡이(`⠿`)를 드래그.
- **우클릭 메뉴**(신호 이름): `위/아래에 신호 추가`, **`CLK 생성`**(주기를 입력받아 클럭 생성 — 0을 `ceil(주기/2)`개, 1을 `floor(주기/2)`개씩 반복. 기본 주기 2), `신호 삭제`.
- **우클릭 메뉴**(셀): `이 열 앞에 열 삽입`, `이 열 삭제`.

### 크기 / 개수

- `행`·`열` 입력칸 또는 격자 **오른쪽/아래/모서리 핸들 드래그**로 신호·시간 개수를 늘리고 줄입니다(셀 크기는 그대로).
- 셀 `W`/`H`, `신호 H`(파형 진폭, 셀 높이와 독립), `선 굵기`를 따로 조절합니다.

## 주석 (화살표 · 점선 · 텍스트)

WaveDrom으로는 표현할 수 없는 화살표·주석을 그릴 수 있습니다.

| 도구 | 설명 |
|------|------|
| **화살표** | 드래그로 그리기. 끝점에 화살촉. |
| **양방향 화살표** | 양쪽 끝에 화살촉. |
| **점선** | 화살촉 없는 점선. |
| **사각형** | 테두리만 있는 직사각형(드래그). **Shift** 를 누르면 **정사각형**. |
| **원/타원** | 테두리만 있는 타원(드래그). **Shift** 를 누르면 **정원**. |
| **T (텍스트)** | 클릭한 위치에 텍스트 추가. |
| **이동** | 화살표/점선/도형/텍스트를 드래그해 이동(끝점·모서리 핸들로 크기·방향 조절). |
| **삭제** | 클릭한 주석 제거. |

- **가운데 텍스트**: 화살표·점선·**사각형·원/타원**은 **가운데에 텍스트**를 넣을 수 있습니다. 그린 직후 바로 입력하거나, `이동` 모드에서 **주석을 더블클릭**해 편집합니다. **줄바꿈(Enter) 가능**, 기본은 빈칸.
- **Shift 고정**: 선은 그리거나 끝점을 옮길 때 **Shift** 로 수평/수직 고정, 도형은 **Shift** 로 정사각형/정원이 됩니다.
- **색 지정**: `선`/`글자` 색을 고르면 **새 주석에 자동 반영**됩니다. 기존 주석은 그대로이며, **`적용`** 을 누른 뒤 주석을 클릭해야 바뀝니다.
- 주석 위치는 **격자 내 비율 좌표**로 저장되어, 셀 크기를 바꿔도 파형을 따라갑니다.

## 캡처 모드 / 이미지 저장

- **`캡처 모드`** 버튼(또는 단축키 `C`)을 누르면 **격자선과 행/열 번호가 사라져** 깔끔한 파형만 남습니다. 이 상태로 화면을 캡처해 이미지로 쓰면 됩니다. 다시 누르면 해제됩니다. (툴바·패널은 그대로 보입니다.)
- **`캡처`** 버튼은 캡처 모드 모양(격자선·번호 없음) 그대로의 파형을 **JPG 이미지로 저장**합니다. 캡처 모드를 켜지 않아도 되며, 신호 이름·데이터 라벨·주석까지 포함됩니다.

## 텍스트 변환

### WaveDrom 텍스트

파형을 바꿀 때마다 **자동으로** WaveDrom 문법이 생성됩니다(읽기 전용). `복사` 버튼으로 클립보드에 복사할 수 있습니다.

```json
{ "signal": [
  { "name": "clk", "wave": "01010101…" },
  { "name": "data", "wave": "x.=.=.x.", "data": ["A0","B1"] }
] }
```

> 참고: 빈 셀은 WaveDrom에서 부동(z)으로 표현됩니다. 주석/화살표는 WaveDrom에 담을 수 없으므로 아래 **자체 JSON**을 사용하세요.

### 자체 JSON (무손실 저장/불러오기)

파형 + 주석 + 모든 설정을 **손실 없이** 보존하는 포맷입니다.

- **JSON 생성**: 현재 상태를 아래 입력창에 출력.
- **JSON 저장**: 파일로 저장(탐색기 창; 단축키 `Ctrl+S`).
- **JSON 열기**: 파일에서 불러오기.
- **파형 생성**: 입력창의 JSON 내용을 그대로 복원.

## 키보드 단축키

| 키 | 동작 |
|----|------|
| `1` / `0` | 1/0 펜 |
| `D` | 데이터 버스 펜 |
| `Z` | High-Z 펜 |
| `X` | Unknown 펜 |
| `S` | shot 펜 |
| `E` | 지움 |
| `C` | 캡처 모드 토글 |
| `Ctrl` + `S` | JSON 저장 |
| `Ctrl` + `Z` | 되돌리기 |
| `Ctrl` + `Y` (또는 `Ctrl`+`Shift`+`Z`) | 다시 실행 |

> 되돌리기/다시 실행은 파형·주석뿐 아니라 **그리드 설정(행·열 개수, 셀 크기, 신호 높이, 선 굵기 등)** 변경에도 적용됩니다.

> 신호 이름·데이터 라벨·텍스트 주석을 **입력하는 중에는 단축키가 동작하지 않습니다.**

## JSON 포맷

```jsonc
{
  "version": "1.0.0",
  "grid": {
    "cols": 20, "cellW": 40, "cellH": 44,
    "signalH": 22, "lineWidth": 2,
    "fillColor": "#add8e6", "textColor": "#000000",   // 데이터 배경/글자 기본색
    "lineColor": "#c0392b", "annoTextColor": "#c0392b" // 주석 선/글자 기본색
  },
  "signals": [
    {
      "name": "clk",
      "cells": [
        { "type": "bit", "value": 1 },                           // 1/0
        { "type": "data", "id": 1, "value": "A0",                // 데이터 버스
          "color": "#add8e6", "textcolor": "#000000" },
        { "type": "z" },                                         // High-Z
        { "type": "x" },                                         // Unknown
        { "type": "empty" }                                      // 빈 칸
      ]
    }
  ],
  "annotations": [
    { "type": "arrow", "from": { "gx": 3, "gy": 1.5 },           // 화살표/점선
      "to": { "gx": 6, "gy": 1.5 },
      "heads": "end",        // "end" | "both" | "none"(점선)
      "dashed": false,
      "color": "#c0392b",    // 선 색
      "textcolor": "#c0392b",// 가운데 글자 색
      "text": "rising" },
    { "type": "shape", "shape": "rect",                          // 사각형/원·타원
      "from": { "gx": 1, "gy": 0.2 }, "to": { "gx": 5, "gy": 1.8 }, // 바운딩 박스 두 모서리
      "color": "#c0392b", "textcolor": "#c0392b", "text": "block" },
    { "type": "text", "at": { "gx": 8, "gy": 0.5 },              // 독립 텍스트
      "text": "note", "color": "#c0392b" }
  ]
}
```

- 셀 `type`: `bit`(`value`: 0/1) · `data`(`id`로 버스 구분, `value`=라벨, `color`/`textcolor`) · `z` · `x` · `empty`.
- 주석 좌표 `gx`/`gy` 는 **격자 단위 비율 좌표**입니다(정수부 = 칸 번호, 소수부 = 칸 안 위치). 셀 크기가 바뀌어도 위치가 유지됩니다.

## 브라우저 호환성

- **Chrome / Edge** (Chromium): 모든 기능 지원. `JSON 저장`이 탐색기 「다른 이름으로 저장」 창으로 열립니다.
- 그 외 브라우저: `JSON 저장`은 일반 다운로드로 대체됩니다(나머지 기능 동일).
